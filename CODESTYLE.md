# Code style

## Namespaces

lower case, e.g. `iganet`, `iganet::utils`.

## Enumerators

lower case, e.g., `activation`.

## Classes

camel case, e.g., `ActivationFunction`.

## Template parameters

camel case, e.g., `BSplineCommon`.

## Type aliases

snake case, e.g., `eval_type`, `boundary_eval_type`.
